# Avionics Embedded Control & Monitoring Simulator

This project is a C/C++ portfolio project that simulates an avionics-style embedded control and health monitoring unit.

It demonstrates:

- Embedded C module design
- C++ application layer architecture
- CAN Bus and ARINC 429 style message parsing
- Watchdog and fault management
- Deterministic state machine design
- Unit testing and static analysis mindset
- DO-178C-inspired requirement traceability and verification documentation

> Note: This is not a certifiable DO-178C project. It is an educational and portfolio-oriented simulation project inspired by safety-critical avionics software development practices.
